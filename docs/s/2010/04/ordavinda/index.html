Orðavinda

My submission to the programming competition Þú átt orðið was the computer game Orðavinda (loosely translated: "Wordwinder"). The basic premise of the competition was to create something interesting upon a newly opened declension database of modern Icelandic language.

The game is based on an older idea and of course a long history of word games. To my pleasant surprise I won the first prize, and judgeing by response, have managed to halt all office work in Iceland.

The game is completely made in HTML and JavaScript but uses Flash if available to play sounds. Anyone curious about the internals should examine the game's GitHub repository. The game is free software under ther terms of the GPL software license.
